# Bulletin Board

## Overview

A simple bulletin board application that allows users to create and view posts.

## Features

*   Create a new post with a title and content.
*   View a list of all posts.
*   **Post Persistence:** Posts are saved in the browser's local storage so they persist after a page refresh.

## Design

*   **Layout:** A simple, clean layout with a form for creating posts and a list of posts below.
*   **Styling:** Basic CSS for readability and a pleasant user experience.
